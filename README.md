## 🤖 Uso de Inteligencia Artificial (Disclaimer)

Para la planificación, estructuración y redacción de la documentación teórica de este curso, se ha contado con el soporte de herramientas de **Inteligencia Artificial Generativa** (como [Gemini y ChatGPT]). 

El uso de estas herramientas se ha gestionado bajo los siguientes criterios:

* **Supervisión docente:** Todo el contenido generado ha sido verificado, corregido y adaptado pedagógicamente por el autor para garantizar su rigor técnico y alineación con el currículo de FP.
* **Código fuente:** Los ejercicios prácticos, scripts de programación y modelos de IA incluidos han sido desarrollados y testeos de forma manual para asegurar su correcto funcionamiento en el aula.
